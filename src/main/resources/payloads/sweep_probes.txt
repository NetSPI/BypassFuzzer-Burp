# - {PATH_FIRST_SEGMENT_UPPERCASE}: original path with the first segment uppercased
# - {PATH_LAST_SEGMENT_UPPERCASE}: original path with the last segment uppercased
# - {PATH_CAPITALIZED}: original path with each segment capitalized

# - {PATH_URL_ENCODE_CHAR_2}: original path with the second encodable character URL encoded
# - {PATH_URL_ENCODE_CHAR_3}: original path with the third encodable character URL encoded
